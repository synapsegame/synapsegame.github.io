v0.0.8

May 24, 2023 · 2 min read

Resources​

  • Resource nodes appear on certain planets and can be mined for their resources
  • Resources are a premium currency used for crafting and item upgrades
  • Enemies also have a chance to drop resources on death. Any resource can drop in this way
  • Resources can be exchanged for energy cells. Currently a flat rate of 1:10. Will update for resource rarity in later update

Crafting​

  • Weapons can be upgraded using resources discovered throughout the universe
  • Crafting is available at any point during planetary exploration
  • An initial set of weapons have upgrades available. More coming in future update

Bosses​

  • Bosses can now spawn on levels preceding extraction
  • Bosses will introduce new and unique mechanics over time
  • First boss has been added at wave 10 with placeholder behavior and animation

Item Types​

  • Added new item types which modify game behavior beyond stat modifications
  • Initial set includes Ion Cannon, MinerBot, Free rerolls (perk and item), Item Store discounts, and chance for double cell pickup
  • Ion Cannon provides aerial support
  • MinerBot collects resources for the player
  • Discounts cannot be discounted
  • Ion Cannon, MinerBot, and reroll items can only be purchased once

Changes

  • Reduced the fire rate on blunderbusses (blunderbi?) and increased the cost
  • Run stats interface updated to group related values from perks and items
  • Movement controls tweaked to improve feel
  • Enemy rarity now included in unit spawn logic
