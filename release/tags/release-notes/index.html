v0.0.6

March 28, 2023 路 2 min read
  • Items now appear in the item store
    • Some placeholder images being used
    • Stats will be adjusted and balanced in time. Initial values in place to confirm proper integration with game logic
  • Item modifiers apply to player stats and the enemy spawner
  • Text popups for damage (units, players) and pickups (cells)
  • Audio for combat, pickups, and some initial background music
  • Particle effects for damage and movement
  • Animations for biological mite units
  • New hero sprite
  • Weapons have kickback on fire
  • Mechanical arms appear to hold more weapons
  • Volume controls in pause menu

Changes

  • Three new enemy types
    • Armored mite (wave 3)
    • Cyber mite (wave 5)
    • Cyber mite with explosive projectiles (wave 8)
  • Enraged units now spawn adjusted color and slight scale increase
  • Enemies move slightly away from player on death (to create knockback visual)
  • Unit stats now scale with level after the unit has appeared for one level previously
  • Character flashes blue if damage done to armor, red if damage done to health
  • Moved the joystick a bit further down the screen and disabled enemy spawn at bottom 20% of screen

Bugfix

  • Fixed bug where the player could reroll if all four items are locked in the item shop
  • Fixed bug where player could buy additional weapons, even if all slots were full. Have not yet implemented logic for upgrading if same rarity.

v0.0.5

March 25, 2023 路 One min read
  • The pause menu now displays release notes for the current version
  • initial set of items now available after each wave
  • Two additional weapons can be equipped (total of four)
  • Death screen now displays your all-time highest wave completed
  • Added spread damage and projectile functionality for shotgun-like weapons

Changes

  • Perks now display with percentages for name and value
  • Weapons rotate to face target before firing
  • Added perk descriptions
  • Less enemies with more health
  • Reduce the enemy movement speed slightly
  • Reduced space-mite size
  • Set the gun pivot point back towards the handle to make the rotation more realistic
  • Display the player cell count in the item store
  • Quite a bit behind the scenes in preparation for non-weapon items in the item store

Bugfix

  • Weapons are purchased even if player does not have enough open slots
  • Cells are not subtracted on item purchase
  • Reroll currently does nothing because there are only four items
  • Locked items not persisting between levels

v0.0.4

March 24, 2023 路 One min read
  • The game now has a name- Synapse!
  • Created an instagram and registered playsynapse.com
  • Randomly spawn health and armor pickups on the level
    • Pickups give cells if health pickup and player has max health
    • Armor pickup always adds armor
    • Pickups can spawn closer to player if less than 5 health

Changes

  • Limited number of corpses on screen to 30
  • Made everything smaller on screen
  • Adjusted hitboxes to be a bit more forgiving for the player
  • Add a pickup spawn boost perk to reduce time between pickup spawns
  • Adjusted fire rate of weapons to make the game a bit more challenging
  • Significantly adjusted the number of enemies that spawn in later waves. You should be able to make it past level 15 now.
  • Updated the application icon and loading screen
  • Units become disinterested if far enough away from player
  • Units will flee if a threshold of units are killed in their proximity
  • 15% chance to spawn enraged units with increased stats and no chance of threat
  • 50% chance of receiving cell pickup at end of level (make a stat that adjusts this)
