v0.0.7

May 24, 2023 Â· 3 min read
  • Available on Android!
  • Added first cosmic blessing encounter (Divine Knowledge)
  • Can now tap anywhere to control movement. Removed joystick
  • Added first-time tutorial popup
  • Added Tendril enemy with new attack type
  • Added Beam Ray weapon with new projectile type (piercing)

Changes

  • Item rarities rendered in item panels
  • Adjusted modifiers of legendary items
  • Factored in item rarity when populating item shop
  • Modifiers now display with styling to indicate if the effect is helpful or harmful for the player
  • Able to view equipped weapons in item screen
  • Added ability to upgrade weapons
  • If background music is enabled, subtle fade out during round end / store display, then fade back in when next round starts
  • Units are now classified by factions, which will allow for future adjustments in gameplay (mechanial damages bio, mechanical partners with AI, cosmic fights all, etc
  • Added subtle shadows to UI elements
  • Added remaining rarities for initial weapons. Balanced values a bit more
  • Adjusted rarity scaling through levels to increase chance of rarer items in latest levels
  • Corrected a sneaky bug in the perk rarity selection logic. Unfortunately for players, rarer perks are now... rarer. Will tune this pending feedback.
  • Player projectiles adjust in scale and color as damage increases
  • Removed caps on enemy damage, health, projectile speed, and movement speed because apparently getting past wave 20 isn't difficult.
  • Added movement effect to units, pickups, and background
  • Attempting to make the movement control panel a bit more obvious
  • Main menu spaceship
  • Haptics on joystick press
  • Removed the text that indicates which slot each item is held in.
  • [[Balance]] The SMG is rather useless when compared to the assault rifle. The fire rate could probably be reduced significantly on the SMG
  • [[Bug]] enemy projectiles are not despawned at the end of the level. add an event listener to the projectiles to check for the level end event, OR, parent them to a collection and then destroy the contents of that collection
  • [[GameFeel]] also, add an attack cooldown randomized interval to spiders so that they're not casting the wave of projectiles like they are in this version
  • [[Balance]] For each level / wave, it would be good to scale the spawn sizes throughout each wave. So when the wave starts, get the maximum spawn size/ from the previous round and the current round, and then using the wave duration, scale the number of units to spawn so that the last 30 seconds are spawning with the full wave capacity.
  • [[Bug]] Synthlegs movement reduced from 2000% to 400%
  • [[GameFeel]] Reduced spider movement speed
  • [[Balance]] Maybe add a stat ceiling for stats that get initialized in the UnitStats method so that units are not scaling uncontrollably in the later levels
  • Armored space mites now have a chance to consume cells, incentivizing player to pick them up
  • Added confirmation double tap to reroll buttons and sell item button
  • [[Interface]] Item modifiers displayed on the death/final screen

New Items

  • armor piercing rounds: +10% damage, -2.5% fire rate
  • Added rare item which provides +5 health
  • Bipod: +10% Attack Range, -5% movement speed
  • Ocular implants: +75 range

Bugfix

  • Try to figure out why loading between scenes is causing some fonts to be discarded
  • The confirm button is not resetting after time. Also need to reset it between rounds
  • Fixed bug where enemy armor was not taken into consideration
